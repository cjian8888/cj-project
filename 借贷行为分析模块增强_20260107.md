### 1. 借贷配对分析（Loan Pairing）

**功能描述**：智能匹配借入和还款记录，计算隐含利率

**核心算法**：

- 在365天时间窗口内匹配收入（借入）和支出（还款）
- 金额匹配容差：±20%（允许利息）
- 自动计算利息率和年化利率
- 排除公共支付平台和公司往来

**风险判定**：

- 年化利率 > 36%：高风险（超过法律红线）
- 年化利率 < 0.1% 且金额 ≥ 5万：高风险（疑似无息借贷）
- 借贷金额 ≥ 10万：高风险（大额借贷）

**示例输出**：

```
借入: 2024-01-15 50,000元
还款: 2024-07-15 52,000元
周期: 182天
年化利率: 8.0%
风险: 正常
```

### 2. 无还款借贷检测（No Repayment Detection）

**功能描述**：发现可能的利益输送 - 大额收入但长期无对应还款

**检测条件**：

- 单笔收入 ≥ 2万元
- 距今时间 ≥ 180天
- 还款比例 < 50%

**风险判定**：

- 金额 ≥ 5万：高风险
- 金额 < 5万：中风险

**典型场景**：

- 领导干部从下属处"借款"长期不还
- 企业主从关联方获得资金无偿使用
- 利用职权获取的隐性收入

**示例输出**：

```
收入: 2023-01-01 100,000元
距今: 1102天
还款比例: 0.0%
风险: 1102天未还款，还款比例仅0.0%
```

### 3. 异常利息检测（Abnormal Interest Detection）

**功能描述**：识别高利贷和疑似无息借贷

**正常利率范围**：4% - 24%（年化）

**异常类型**：

| 年化利率 | 异常类型 | 风险等级 |
|---------|---------|---------|
| > 36% | 超高利贷（超过法律红线） | 高 |
| 24% - 36% | 高利贷 | 高 |
| < 4% 且金额≥5万 | 疑似无息借贷（可能利益输送） | 高 |
| < 0.1% 且金额≥5万 | 疑似无息借贷（可能利益输送） | 高 |

**法律依据**：

- 《最高人民法院关于审理民间借贷案件适用法律若干问题的规定》
- 年利率超过36%的部分无效

**示例输出**：

```
金额: 30,000元
年化利率: 80.2%
异常类型: 超高利贷（超过36%红线）
```

### 4. 借贷网络分析（Loan Network Analysis）

**功能描述**：构建借贷关系图谱，识别借贷中心节点

**网络元素**：

- **节点（Nodes）**：核心人员 + 关联对手方
- **边（Edges）**：借贷关系（带金额和次数）
- **中心节点（Hubs）**：与3人及以上有借贷关系的节点

**应用场景**：

- 识别"掮客"角色（多次充当中间人）
- 发现借贷链条（A→B→C）
- 分析资金流向网络

**示例输出**：

```

```
