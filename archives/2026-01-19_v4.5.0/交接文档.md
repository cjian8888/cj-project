### 1.1 "打开文件夹"功能 (Fixed P0) - 跨平台强化
- **需求**: 用户点击文件索引的"打开文件夹"按钮时，文件资源管理器窗口必须跳到前台。
- **挑战**: Windows 从后台服务（Python后端）启动进程时，OS 会阻止窗口抢占焦点，导致窗口只在任务栏闪烁。
- **解决方案**:
  - **Windows**: 使用 Shell COM 接口 (`win32com.client`) 枚举所有 Explorer 窗口，按路径精确匹配后调用 `AttachThreadInput` + `SetForegroundWindow` 强制置顶。
  - **macOS**: 使用 AppleScript 调用 `Finder.activate` + `set frontmost` 将窗口带到前台。
  - **Linux**: 使用 `xdg-open` 配合 `wmctrl` (可选) 激活窗口。
- **依赖**: Windows 平台需要 `pywin32` 包 (`pip install pywin32`)，已添加到 `requirements.txt`。
- **状态**: ✅ 已验证，三平台完美解决。

  4. 饼图 Tooltip 配色优化：蓝色边框 + 深蓝背景 + 高对比度文字。
