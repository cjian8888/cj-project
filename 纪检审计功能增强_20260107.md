## 一、本次新增功能模块

### 1. 关联方资金穿透分析器

**文件**: `related_party_analyzer.py`

| 功能 | 说明 |
|------|------|
| 直接资金往来 | 检测核心人员之间的直接转账记录 |
| 第三方中转检测 | A→C→B模式，自动排除支付宝/财付通等公共支付平台 |
| 资金闭环检测 | 检测A→B→C→A等资金回流模式 |
| 高频中间人识别 | 标记多次充当中间人的账户（掮客嫌疑） |

**过滤规则**:

- 最低金额门槛：1万元
- 排除公共支付平台：财付通、支付宝、微信支付、京东支付等
- 排除电商平台：淘宝、天猫、京东、美团、饿了么等
- 排除公共服务：移动/联通/电信、电力、水务等

---

### 2. 特殊时段异常交易检测

**文件**: `suspicion_detector.py` (新增函数)

| 函数 | 功能 |
|------|------|
| `detect_cash_timing_pattern()` | 关联人现金存取时间点配对 |
| `detect_holiday_transactions()` | 节假日/周末/非工作时间大额交易 |
| `detect_amount_patterns()` | 拆分规避/整数偏好/吉利数尾号检测 |

---

### 3. 多源数据交叉碰撞分析器

**文件**: `multi_source_correlator.py`

| 功能 | 数据源 |
|------|--------|
| 出行同行人资金碰撞 | 航班同行人、铁路同行人 |
| 同住宿人资金碰撞 | 公安部同住宿数据 |
| 快递联系人资金碰撞 | 国家邮政局快递信息 |

---
