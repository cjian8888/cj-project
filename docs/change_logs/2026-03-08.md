# 2026-03-08 变更记录

## 变更目标
- 提升风险情报识别的准确性与可解释性。
- 修复 HTML 报告数据缺失与模块开关不生效问题。
- 打通前端报告配置（核查对象/阈值/模块）到后端生成算法。
- 修复数据概览「风险研判」金额统计为 0 的前端展示问题。

## 核心改动摘要
- 后端分析链路增强：补齐资金穿透、ML 分析、线索聚合入参，优化疑点检测主流程与兼容字段。
- 现金碰撞规则优化：默认不将“同一主体自碰撞”作为主风险碰撞输出，降低误报。
- 报告生成增强：支持运行时 `thresholds/sections`，并真正作用于大额转账/现金分析和 HTML 模板显示。
- 前端适配：统一 API 地址来源，状态字段兼容，KPI 风险金额聚合逻辑修复。
- 报告模板修复：`report_v3` 按模块开关动态显示 summary/assets/risks，避免无效配置。
