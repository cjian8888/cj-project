### 2.4 【中优先级】人员单位/职务信息展示逻辑问题

**后端计算**: ✅ 已从户籍数据获取  
**方法**: `_build_personal_background()` (line 6093)  
**模板使用**: ⚠️ 有条件显示（导致信息隐藏）

**问题分析**:
```python
# 后端已获取
def _build_personal_background():
    employer = huji_info.get("从业单位", "") or "【待调取】"
    return {
        "employer": employer,  # 可能有真实值
        "current_position": "【待调取】",  # 确实需要人事档案
        "entry_date": "【待调取】",  # 确实需要人事档案
    }

# 但模板逻辑是：
{% if bg.employer != '【待调取】' %}进入{{ bg.employer }}工作{% endif %}
```

**问题**: 
1. 即使 employer 有真实值，如果等于 '【待调取】' 字符串，也不会显示
2. 职务(entry_date)和职位(current_position)标记为"【待调取】"，但未说明数据来源

**应展示内容**:
- 从业单位（从户籍数据获取）
- 职务/职级（标注数据来源：人事档案待调取）
- 入职时间（标注数据来源：人事档案待调取）
