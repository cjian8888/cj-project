### 任务6：调整风险等级判定逻辑

#### 当前逻辑
所有异常都标注为"【低风险】"

#### 整改后逻辑
```python
def determine_risk_level(salary_ratio, expense_income_ratio):
    """
    工资占比 + 支出收入比 → 风险等级
    """
    if salary_ratio < 0.2 or expense_income_ratio > 1.5:
        return "high", "高风险"
    elif salary_ratio < 0.5 or expense_income_ratio > 1.2:
        return "medium", "中风险"
    else:
        return "low", "低风险"
```

#### 整改后效果
- 施灵（28.6% + 支出>收入）: 中风险
- 施育（21.7%）: 高风险
- 滕雳（64.3%）: 低风险
