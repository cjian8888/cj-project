# 家庭与个人分析报告改进计划 v2.0

> **状态**: 📝 制定中  
> **制定时间**: 2026-02-12  
> **优先级**: 高（影响报告质量的核心问题）

---

## 一、现状问题诊断

### 1.1 家庭概况问题

**当前显示**:
```
家庭概况：施灵家庭共3人，家庭总资金流入8442.87万元，流出8306.84万元。
其中工资收入0.00万元，占比0.0%。
```

**问题分析**:

| 问题 | 严重程度 | 说明 |
|-----|---------|------|
| 使用流水数据 | 🔴 严重 | 8442.87万是总流水，非真实收入 |
| 工资收入为0 | 🔴 严重 | 实际三人工资合计1088.97万 |
| 无资产快照 | 🔴 严重 | 缺失房产、车辆、存款、理财 |
| 内部转账重复 | 🔴 严重 | 家庭成员互转被重复计算 |

**数据对比**:

| 成员 | 总流水(万) | 真实收入(万) | 虚高倍数 |
|-----|-----------|-------------|---------|
| 施灵 | 6903.58 | 2234.77 | 3.1x |
| 滕雳 | 1104.98 | 537.71 | 2.1x |
| 施承天 | 434.31 | 305.52 | 1.4x |
| **家庭合计** | **8442.87** | **3078.00** | **2.7x** |

### 1.2 个人分析同样问题

**个人章节同样存在**:
- 使用 `total_income`（流水）而非 `real_income`（真实收入）
- 资产盘点不完整
- 缺少收入结构分析

---

## 二、改进目标

### 2.1 短期目标（当前版本）

1. **数据准确性** - 使用真实收入替代流水
2. **资产完整性** - 整合已有外部数据
3. **内部转账剔除** - 避免重复计算
4. **工资收入修正** - 正确汇总工资数据

### 2.2 中期目标（下个版本）

5. **收入结构分析** - 工资/经营/投资/其他分类
6. **资金往来透明** - 大额对手方分析
7. **风险指标预警** - 异常指标自动标记
8. **可视化图表** - 收入结构饼图、趋势图
