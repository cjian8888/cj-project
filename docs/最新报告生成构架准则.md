## 七、核心实现规则

### 7.1 银行账户过滤规则
```python
ACCOUNT_EXCLUDE_KEYWORDS = ["基金", "理财", "证券", "期货", "信托", "资管", "托管"]

def is_real_bank_account(bank_name, account_number, card_type):
    # 排除关键词
    for kw in ACCOUNT_EXCLUDE_KEYWORDS:
        if kw in bank_name or kw in card_type:
            return False
    
    # 账号长度校验（16-19位）
    if account_number:
        digits = len(account_number.replace(" ", ""))
        if digits < 16 or digits > 19:
            return False
    
    return True
```

### 7.3 年度工资统计规则

def aggregate_family_yearly_salary(members):
    yearly_data = {}
    
    for name in members:
        profile = profiles.get(name, {})
        yearly_salary = profile.get("yearlySalary", {}).get("yearly", {})
        
        for year, year_data in yearly_salary.items():
            if year not in yearly_data:
                yearly_data[year] = {}
            yearly_data[year][name] = year_data.get("total", 0)
    
    return FamilyYearlySalaryList(yearly_data)
