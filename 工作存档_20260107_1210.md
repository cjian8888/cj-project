## 📝 技术要点

### 风险判定规则

```python
# 借贷配对风险
年化利率 > 36%  → 高风险（超法律红线）
年化利率 < 0.1% 且 金额≥5万 → 高风险（疑似无息借贷）
借贷金额 ≥ 10万 → 高风险

# 无还款借贷风险
单笔收入 ≥ 2万 + 距今≥180天 + 还款<50% → 疑似利益输送
金额 ≥ 5万 → 高风险
```

### 排除规则

```python
# 通用排除
同名对手方（自己转自己）
核心人员间转账（已在借贷分析）
第三方支付平台（支付宝/微信等）
银行系统账户（代付/代收/业务专户）

# 工资排除
SALARY_STRONG_KEYWORDS
KNOWN_SALARY_PAYERS
USER_DEFINED_SALARY_PAYERS
研究所/研究院
```

## 🚀 重大突破：双引擎智能审计

### 1. 机器学习模块重构 (ml_analyzer.py)

从单纯的统计异常检测升级为 **“统计 + 图论”双引擎**，在不引入 `sklearn` 和 `networkx` 的前提下实现了高级团伙挖掘。

| 引擎类型 | 实现类名 | 核心算法 | 负责功能 |
| :--- | :--- | :--- | :--- |
| **统计引擎** | `LightweightAnomalyDetector` | Z-Score + IQR Ensemble | 发现单笔异常（金额偏离、深夜交易、整额特征） |
| **图引擎** | `GraphCommunityDetector` | 手写 Tarjan 强连通分量 (SCC) | 发现资金闭环、高频资金圈子 |

### 2. 团伙挖掘成果

- **算法实现**：纯 Python 实现 Tarjan 算法，寻找强连通分量。
- **去噪处理**：增加了**超级节点黑名单**（支付宝、微信、银联等），有效剔除了“因为都用支付宝所以被连在一起”的虚假团伙，使结果更具情报价值。
- **验证结果**：
  - 成功挖出一个 **89人、5300万** 的资金闭环网络。
  - 发现跨地域异常连接：上海/浙江/安徽等多地**渔具店**与核心人员的资金闭环。
  - 发现**贵州系**企业集群与个人的紧密资金往来。
